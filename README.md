# 📊 Adult Dataset Visualization & Dashboard

## 📌 Project Overview

This project focuses on data visualization and dashboard creation using the Adult Income dataset. The goal is to explore patterns and relationships between features such as age, education, working hours, and income.

## ⚙️ Technologies Used

* Python
* Pandas
* Plotly
* Matplotlib & Seaborn
* Streamlit (for dashboard)

## 📂 Dataset

The dataset contains demographic and work-related information used to predict whether a person earns more than 50K per year.

## 📊 Visualizations

The following visualizations were created:

* Age distribution (Histogram)
* Education distribution (Bar chart)
* Age vs Working Hours (Scatter plot)
* Income distribution (Pie chart)
* Income vs Education (Grouped chart)
* Correlation heatmap

## 📈 Key Insights

* Higher education levels are associated with higher income.
* Individuals working more hours per week tend to earn more.
* Age has a moderate relationship with working hours and income.

## 🖥️ Dashboard

An interactive dashboard was created using Streamlit, allowing users to:

* Filter data by age
* Explore different visualizations dynamically

## 🚀 How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Run the dashboard:

```bash
streamlit run app.py
```

## 🎯 Conclusion

This project demonstrates how data visualization can help uncover meaningful insights and support decision-making.
